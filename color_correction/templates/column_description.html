Column Descriptions

Detection & Correction Process

  • Index: Row number identifier
  • Detection Method: Method used for color chart detection
  • Detection Parameters: Parameters used in the detection process
  • Correction Method: Method used for color correction
  • Correction Parameters: Parameters used in the correction process

Images

  • Drawed Preprocessing Input/Reference: Debug visualization of preprocessing step
  • Color Patches - Before/After: Color patches comparison before and after correction
  • Input/Corrected Image: Original and color-corrected images

Color Difference Metrics (ΔE)

  • Patch ΔE (Before): Color difference statistics for patches before correction
  • Patch ΔE (After): Color difference statistics for patches after correction
  • Image ΔE: Overall image color difference statistics
  • Statistics include: Minimum, Maximum, Mean, and Standard Deviation values
